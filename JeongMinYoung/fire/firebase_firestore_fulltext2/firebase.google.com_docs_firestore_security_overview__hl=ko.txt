## 본문
Cloud Firestore는 사용하는 클라이언트 라이브러리에 따라 서로 다른 두 가지 메서드를 통해 강력한 액세스 관리 및 인증을 제공합니다.

모바일 및 웹 클라이언트 라이브러리의 경우 Firebase Authentication 및 Cloud Firestore Security Rules을 사용하여 서버리스 인증, 승인, 데이터 유효성 검사를 처리합니다. Cloud Firestore Security Rules으로 Android, Apple, 웹 클라이언트 라이브러리의 데이터를 보호하는 방법을 알아보세요.

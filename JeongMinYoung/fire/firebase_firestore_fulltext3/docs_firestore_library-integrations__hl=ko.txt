## 서문
Cloud Firestore는 본 문서에서 다룬 클라이언트 및 서버 라이브러리 외에도 다양한 오픈소스 라이브러리와의 통합이 가능합니다. 이 같은 통합은 주로 Cloud Firestore를 사용하는 개발자가 자주 이용하는 프레임워크로 이를 가져오는 과정에서 이루어집니다.

Cloud Firestore팀과의 협력을 통해 구현된 통합은 다음과 같습니다.

## AngularFire
플랫폼: 웹

AngularFire는 Firebase용 Angular 라이브러리입니다. AngularFire 및 Cloud Firestore를 사용하면 데이터를 문서 및 컬렉션에 추가하고 데이터를 쿼리하며 오프라인 데이터 액세스를 활성화할 수 있습니다.

## FirebaseUI
플랫폼: Android, iOS

Android 및 iOS용 FirebaseUI 라이브러리를 사용하면 Cloud Firestore 데이터베이스의 데이터를 앱의 UI에 바인딩할 수 있습니다.

## React Native Firebase
플랫폼: Android, Apple

React Native Firebase는 공식 지원되는 Firebase용 React Native 라이브러리입니다. 단, Firebase 지원이 제공되지 않습니다 React Native에서 Cloud Firestore 시작하기에서 React Native 및 Cloud Firestore로 가능한 작업에 대해 자세히 알아보세요.

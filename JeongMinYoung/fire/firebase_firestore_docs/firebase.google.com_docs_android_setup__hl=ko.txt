2023년 10월에 Kotlin 확장 프로그램(KTX) API가 각각의 기본 모듈에 추가되었습니다. 예를 들어 firebase-perf-ktx의 모든 API는 com.google.firebase.perf 패키지 아래 firebase-perf에 추가되었습니다.
 이 변경사항으로 인해 Kotlin 개발자는 이제 KTX 모듈 대신 기본 모듈을 사용해야 합니다(Firebase BoM v32.5.0 이상 또는 BoM v32.5.0 이상에 나열된 기본 모듈 버전을 사용하는 경우).

이전에 출시된 KTX 모듈에서 KTX API를 사용하는 경우 대신 기본 모듈에서 KTX API를 사용하도록 앱을 마이그레이션하는 것이 좋습니다. 자세한 내용은 이 이니셔티브에 대한 FAQ를 참조하세요.

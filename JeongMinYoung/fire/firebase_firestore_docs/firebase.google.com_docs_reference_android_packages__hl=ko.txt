## 서문
In July 2025, we stopped releasing new versions of KTX modules, and we removed the KTX libraries from the Firebase Android BoM (v34.0.0).

Back in October 2023, the Kotlin extensions (KTX) APIs were added to their respective main modules. For example, all the APIs from firebase-perf-ktx were added to firebase-perf under the com.google.firebase.perf package. This change means that Kotlin developers should now depend on the main modules instead of the KTX modules (when using Firebase BoM v32.5.0+ or main module versions listed in BoM v32.5.0+).

If you use KTX APIs from the previously released KTX modules, we strongly recommend that you migrate your app to use KTX APIs from the main modules instead. For details, see the FAQ about this initiative.

## 본문
Deletes a database.

DELETE https://firestore.googleapis.com/v1/{name=projects/*/databases/*}

The URL uses gRPC Transcoding syntax.

The request body must be empty.

If successful, the response body contains an instance of Operation.

Requires one of the following OAuth scopes:

https://www.googleapis.com/auth/datastore

https://www.googleapis.com/auth/cloud-platform

For more information, see the OAuth 2.0 Overview.

### 표
| Parameters | |
| --- | --- |
| Parameters | |
| name | string Required. A name of the form projects/{projectId}/databases/{databaseId} |

| Parameters | |
| --- | --- |
| Parameters | |
| etag | string The current etag of the Database. If an etag is provided and does not match the current etag of the database, deletion will be blocked and a FAILED_PRECONDITION error will be returned. |

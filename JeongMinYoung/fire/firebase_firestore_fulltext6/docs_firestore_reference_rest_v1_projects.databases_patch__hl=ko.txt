## 본문
Updates a database.

PATCH https://firestore.googleapis.com/v1/{database.name=projects/*/databases/*}

The URL uses gRPC Transcoding syntax.

The request body contains an instance of Database.

If successful, the response body contains an instance of Operation.

Requires one of the following OAuth scopes:

https://www.googleapis.com/auth/datastore

https://www.googleapis.com/auth/cloud-platform

For more information, see the OAuth 2.0 Overview.

### 표
| Parameters | |
| --- | --- |
| Parameters | |
| database.name | string The resource name of the Database. Format: projects/{project}/databases/{database} |

| Parameters | |
| --- | --- |
| Parameters | |
| updateMask | string (FieldMask format) The list of fields to be updated. This is a comma-separated list of fully qualified names of fields. Example: "user.displayName,photo". |

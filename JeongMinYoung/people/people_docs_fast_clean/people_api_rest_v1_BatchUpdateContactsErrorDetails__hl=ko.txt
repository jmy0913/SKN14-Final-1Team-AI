키는 BatchUpdateContactsRequest.contacts의 리소스 이름 / 연락처 ID입니다.
값은 상태입니다. google.rpc.Status.details에는 연락처의 오류 목록이 포함됩니다.

"key": value 쌍 목록을 포함하는 객체입니다. 예: { "name": "wrench", "mass": "1.3kg", "count": "3" }

연락처 일괄 삭제 연락처가 아닌 데이터는 삭제되지 않습니다.

지연 시간 및 실패 증가를 방지하려면 동일한 사용자에 대한 변형 요청을 순차적으로 전송해야 합니다.

HTTP 요청

POST https://people.googleapis.com/v1/people:batchDeleteContacts

URL은 gRPC 트랜스코딩 구문을 사용합니다.

요청 본문

요청 본문에는 다음과 같은 구조의 데이터가 포함됩니다.

JSON 표현
{
  "resourceNames": [
    string
  ]
}
필드
resourceNames[]

string

필수 항목입니다. 삭제할 연락처의 리소스 이름입니다. 반복 가능합니다. 단일 요청에 최대 500개의 리소스 이름을 허용합니다.

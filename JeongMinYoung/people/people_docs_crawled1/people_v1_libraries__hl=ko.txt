People API는 HTTP 및 JSON에 기반하므로 모든 표준 HTTP 클라이언트는 요청을 보내고 응답을 파싱할 수 있습니다.
그러나 Google API 클라이언트 라이브러리는 HTTP를 통한 API 요청에 대한 통합 및 향상된 보안 클라이언트 라이브러리는 다양한 프로그래밍 언어 사전 정의된 임곗값을 사용하면 응답을 파싱합니다
시작하려면 개발에 사용할 프로그래밍 언어를 선택하세요.

Ruby용 Google API 클라이언트 라이브러리의 시작 가이드 읽기
People API용 API 탐색기를 사용하여 브라우저에서 이 API와 상호작용
google-api-client gem 설치
시스템에 따라 명령어 앞에 sudo를 추가해야 할 수 있습니다.
Ruby용 Google API 클라이언트 라이브러리를 설치하지 않았으면 RubyGems를 사용하여 설치합니다.
gem install google-api-client
이미 gem을 설치했다면 최신 버전으로 업데이트합니다.
gem update -y google-api-client
Ruby용 Google API 클라이언트 라이브러리 시작하기
첫 번째 요청을 만드는 방법은 시작 가이드를 참조하세요.
People API에 액세스하는 다른 방법
다음 표에는 People API에 액세스하는 다른 방법이 나와 있습니다.
액세스 방법 설명
API 탐색기 브라우저에서 바로 Google API를 사용해 볼 수 있는 대화형 도구
Apps Script Google 제품 및 서드 파티 서비스 전반에서 작업을 자동화하는 JavaScript 클라우드 스크립팅 언어입니다.

# REST Resource: people.connections
bookmark_border

## 본문

리소스: Person
인증된 사용자의 연락처 및 프로필 데이터 등 다양한 데이터 소스에서 병합된 사람에 대한 정보입니다.
대부분의 필드에는 여러 개의 항목이 포함될 수 있습니다. 필드의 항목에는 순서가 보장되지 않지만 비어 있지 않은 각 필드에는 metadata.primary가 true로 설정된 필드가 정확히 한 개 있습니다.

필드

string

string

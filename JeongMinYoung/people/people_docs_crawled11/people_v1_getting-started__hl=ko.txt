첫 번째 클라이언트 애플리케이션의 코딩을 시작하기 전에 다음 세 가지 작업을 해야 합니다.

아직 수행하지 않은 경우 이 가이드에서는 이러한 작업을 안내합니다.

1. Google 계정 만들기

Google API 콘솔에서 프로젝트를 생성하려면 Google 계정이 필요합니다. 계정이 이미 있다면 준비가 된 것입니다. 테스트 목적으로 별도의 Google 계정을 만들 수도 있습니다.

2. 클라이언트용 프로젝트 만들기

People API에 요청을 보내려면 먼저 Google에 클라이언트에 관해 알리고 API에 대한 액세스를 활성화해야 합니다. Google API 콘솔을 사용하여 설정 및 API 액세스 정보를 모아 이름을 붙인 프로젝트를 만들고 애플리케이션을 등록하면 됩니다.

People API를 사용하려면 먼저 설정 도구를 사용해야 합니다. 이 도구는 Google API 콘솔에서 프로젝트를 만들고, API를 사용 설정하고, 사용자 인증 정보를 만드는 방법을 안내합니다.

3. 앱 설정

지원되는 언어 중 하나를 사용하는 경우 클라이언트 라이브러리 중 하나를 사용하는 것이 좋습니다. API에 요청을 수행하려면 먼저 승인을 설정해야 합니다.

다양한 언어로 연락처 목록을 가져오는 방법을 보여주는 빠른 시작 샘플 코드를 살펴보세요.

연락처 그룹 리소스 이름을 지정하여 인증된 사용자가 소유한 특정 연락처 그룹을 가져옵니다.

GET https://people.googleapis.com/v1/{resourceName=contactGroups/*}

URL은 gRPC 트랜스코딩 구문을 사용합니다.

string

필수 항목입니다. 가져올 연락처 그룹의 리소스 이름입니다.

integer

선택사항입니다. 반환할 최대 구성원 수를 지정합니다. 설정하지 않으면 기본값은 0으로, 멤버 0개를 반환합니다.

string (FieldMask format)

선택사항입니다. 그룹에서 반환되는 필드를 제한하는 필드 마스크입니다. 설정하지 않거나 비워 둘 경우 기본값은 metadata, groupType, memberCount, name입니다. 유효한 필드는 다음과 같습니다.

clientData

groupType

memberCount

metadata

이름

요청 본문은 비어 있어야 합니다.

성공한 경우 응답 본문에 ContactGroup의 인스턴스가 포함됩니다.

다음 OAuth 범위 중 하나가 필요합니다.

https://www.googleapis.com/auth/contacts

https://www.googleapis.com/auth/contacts.readonly

자세한 내용은 승인 가이드를 참조하세요.

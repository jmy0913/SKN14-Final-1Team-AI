'기타 연락처' 복사 사용자의 'myContacts'에 있는 새 연락처 그룹

지연 시간 및 실패 증가를 방지하려면 동일한 사용자에 대한 변형 요청을 순차적으로 전송해야 합니다.

POST https://people.googleapis.com/v1/{resourceName=otherContacts/*}:copyOtherContactToMyContactsGroup

URL은 gRPC 트랜스코딩 문법을 사용합니다.

요청 본문에는 다음과 같은 구조의 데이터가 포함됩니다.

성공한 경우 응답 본문에 Person의 인스턴스가 포함됩니다.

다음 OAuth 범위 중 하나가 필요합니다.

https://www.googleapis.com/auth/contacts.other.readonly

https://www.googleapis.com/auth/contacts

자세한 내용은 승인 가이드를 참조하세요.

| 필드 | |
| copyMask | string (FieldMask format) 필수 항목입니다. 새 연락처로 복사할 필드를 제한하는 필드 마스크입니다. 유효한 값은 다음과 같습니다. emailAddresses 이름 phoneNumbers |
| readMask | string (FieldMask format) 선택사항입니다. 반환되는 사람의 필드를 제한하는 필드 마스크입니다. 여러 필드는 쉼표로 구분하여 지정할 수 있습니다. 설정하지 않은 경우 기본값은 메타데이터 및 멤버십 필드가 있는 복사 마스크입니다. 유효한 값은 다음과 같습니다. addresses ageRanges 전기 생일 calendarUrls clientData coverPhotos emailAddresses 이벤트 externalIds 성별 imClients 관심분야 locales 위치 멤버십 메타데이터 miscKeywords 이름 닉네임 직업 organizations phoneNumbers 사진 관계 sipAddresses 기술 URL userDefined |
| sources[] | enum (ReadSourceType) 선택사항입니다. 반환할 소스 유형의 마스크입니다. 설정하지 않으면 기본값은 READ_SOURCE_TYPE_CONTACT 및 READ_SOURCE_TYPE_PROFILE입니다. |
